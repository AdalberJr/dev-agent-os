# Handoff Protocol

## Ziel
Übergaben zwischen Rollen klar, knapp und überprüfbar machen.

## Pflichtfelder jeder Übergabe

- Kontext
- Ziel
- Scope
- bekannte Risiken
- gewünschter Output
- Done-Kriterium
- offene Fragen

## Minimales Übergabeformat

### Von
Welche Rolle übergibt?

### An
Welche Rolle übernimmt?

### Kontext
Worum geht es?

### Ziel
Was soll erreicht werden?

### Scope
Was ist drin und was nicht?

### Risiken / Hinweise
Was ist sensibel, kritisch oder unklar?

### Gewünschter Output
Welches Dokument oder Ergebnis wird erwartet?

### Done wenn
Wann gilt die Aufgabe als sauber erledigt?

## Qualitätsregel
Wenn eine Übergabe unklar ist, wird nicht geraten. Dann wird präzisiert.
