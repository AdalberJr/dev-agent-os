# Security Checklist

- [ ] Server-seitige Auth geprüft
- [ ] Rollen / Rechte geprüft
- [ ] Input-Validierung vorhanden
- [ ] Keine Secrets im Client / Repo
- [ ] Sensible Logs geprüft
- [ ] Riskante Endpunkte identifiziert
- [ ] Öffentliche vs private Daten sauber getrennt
- [ ] Restrisiko dokumentiert
